# tidyverse-gifs

Provides an example of how to create a gif using:

`flair` --> `xaringan` + `pagedown` --> `magick`

**Steps:**

1. Create slides (e.g. "xaringan-slides-example.Rmd")
2. Render slides into a gif (e.g. "render-gif.R")
